
# Shortcuts for development, distribution

## build maas mirror
```shell
cd TensorRT-LLM
./maas/scripts/build_image.sh
```


## Dev Install on L40
```shell
cd TensorRT-LLM
./maas/scripts/install.sh

```

## Distribute on SM80(A30), SM89(L40), SM90(H20), SM120(5090)
```shell
cd TensorRT-LLM
./maas/scripts/distribute.sh

```
